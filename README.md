## Frontend Challenge
Frontend for a project management app with create, update, delete, search and pagination. Responsive design.

**Live: https://esto-es-challenge.vercel.app/**

### Used libraries:
- React.js
- Next.js
- useSWR (data fetching)
- Styled components
- TailwindCSS
- formik+yup (form validation)

### How to run locally:
1) Clone the repo
2) npm install
3) npm run dev
